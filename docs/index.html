> PROJECT OVERVIEW / 项目简介

📖 Introduction

NightX-OSS is an open-source, passion-driven continuation project for Minecraft 1.8.9. A passionate tribute keeping the fire burning with new features and stability fixes.

NightX-OSS 是一个由纯粹热爱驱动的 Minecraft 1.8.9 开源延续项目，旨在延续经典的生命力，并为所钟爱的架构注入现代 Kotlin 的硬核动力。

🌟 Our Story & Motivation / 项目初衷

我的故事始于作为 NightX-Client B45 的一名普通玩家。从第一次加载它开始，我就深深地爱上了这款客户端流畅的体验、独特的视觉设计与极佳的操控感。

在痴迷的驱动下，我深入探索了 1.8.9 开源社区底层生态，研究了 LiquidBounce、LiquidBounce+ 和 FDP Client 等先驱项目。看到发包层面开发的无限可能后，我决定将这份热爱付诸行动，衍生制作出了 NightX-OSS。
